## Funcionalidades

- **Detecção de Avarias**: Identificação e classificação de danos na carroceria, arranhões, amassados e irregularidades.
- **Relatórios Detalhados**: Geração de relatórios com resumo das avarias detectadas e recomendações para reparo.
- **Alertas em Tempo Real**: Emissão de alertas sobre avarias detectadas.
- **Níveis de Acesso**: Diferentes permissões de usuário para seguradoras e estacionamentos.
- **Monitoramento Contínuo**: Ajuste de algoritmos para garantir precisão na detecção.
